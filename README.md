# 🌱 Smart Plant System Ultimate

An advanced AI-powered Smart Plant Monitoring System that integrates Computer Vision, IoT, Big Data Analytics, Search Engine, and Generative AI into one unified platform.

---

## 🚀 Overview

This project combines multiple technologies into a single system:

- 📸 AI Image Analysis – Detect plant diseases from leaf images  
- 📡 IoT Monitoring – Track real-time environmental data  
- 📊 Big Data Analytics – Analyze sensor data using Spark  
- 🔍 Search Engine + RAG – Query scientific articles with AI  
- 📑 PDF Reports – Generate analytics reports  
- 💬 AI Chatbot – Interactive assistant powered by Gemini  

---

## 🧠 System Architecture

The system consists of 7 main modules:

1. Image AI (Computer Vision)  
2. IoT Data Collection & Monitoring  
3. Big Data Processing (Spark + Firebase)  
4. Search Engine + RAG Pipeline  
5. Dashboard UI (Gradio)  
6. PDF Report Generator  
7. AI Chatbot

## 📦 Features

### 📸 Plant Disease Detection
- Upload a leaf image  
- AI model detects disease or healthy state  
- Generates care recommendations using Gemini  

---

### 📡 IoT Real-Time Monitoring
Tracks:
- Temperature 🌡️  
- Humidity 💧  
- Soil Moisture 🌿  

Features:
- Parallel data fetching  
- Live updates  
- Dashboard visualization  

---

### 📊 Big Data Analytics
- Min / Max (MapReduce requirement)  
- Average / Standard Deviation  
- Anomaly Detection (3σ rule)  
- Stored in Firebase  

---

### 🔍 Search Engine + RAG
- Custom search engine (inverted index)  
- Tokenization + stemming  
- Ranking by relevance  
- Gemini generates answers from retrieved context  

---

### 🎛️ Dashboard
- Live sensor values  
- Trend visualization  
- System status  

---

### 📑 PDF Report Generator
- Generates professional reports  
- Includes:
  - Big Data analytics  
  - Recent IoT data  

---

### 💬 AI Chatbot
- Answers questions about:
  - Plant health  
  - Sensor data  
  - System functionality
